<strong>실행 순서</strong>를 따라가 보면 가장 잘 이해돼요.

  </div>

## 헷갈리는 순서 퀴즈

```js
console.log('1');
setTimeout(() => console.log('2'), 0);
Promise.resolve().then(() => console.log('3'));
console.log('4');
```

점만 기억해도 이 종류의 문제는 대부분 풀립니다.
